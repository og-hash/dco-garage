Switch Builder - Wiping/Rebuild ACCTON Red 7772 Console with one of these baudrates (115200 or 9600)

Interrupt the boot procedure when the prompt says:
TYPE 123 [ENTER] to STOP autoboot

=>env default -f -a
=>saveenv
=>reset

Once the device boots, it will show a prompt from ONIE (Installer Mode Enabled)
Use the following command to display partitions:
=>at /proc/mtd

Look for the line where it’s written "open" and note the "mtd" number.
[example]
(...)
** Installer Mode Enabled **
ONIE:/ # cat /proc/mtd
dev: size erasesize name
mtd0: 00020000 00020000 "RCW"
mtd1: 00020000 00020000 "hw-info"
mtd2: 02000000 00020000 "onie"
mtd3: 02000000 00020000 "diag"
mtd4: 03ec0000 00020000 "open"
mtd5: 00020000 00020000 "Fman-FW"
mtd6: 00020000 00020000 "uboot-env"
mtd7: 000c0000 00020000 "uboot"
ONIE:/ #

On the above example, the mtd number is 4. Adjust the command below accordingly:
=>flash_erase /dev/mtd4 0 0

After erasure reaches 100%, run:
=>Reboot

After reboot, the device should appear on Switchbuilder.
